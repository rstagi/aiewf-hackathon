# AIEWF Hackathon 2026

Project built for the [2026 AI Engineer World's Fair Hackathon](https://cerebralvalley.ai/e/aiewf-hackathon-2026).

## Theme

Building in one of the hackathon's three required themes:

- **Continual Learning** — LLM systems that continuously improve from real-world use (memory, user feedback, prompt optimization, self-reflection, toolkit expansion).
- **The Self-Improvement Stack** — infrastructure to continuously evaluate, monitor, and upgrade AI systems (evals, observability, deployment loops, model routing, automated experimentation).
- **Recursive Intelligence** — early forms of recursive/self-directed intelligence where models help build themselves.

## Status

🚧 Work in progress — building during the hackathon.
